
Ova aplikacija omogućava analizu dokumenata korišćenjem LDA (Latent Dirichlet Allocation) modela za topic modeling, implementirana je u Streamlitu. Korisnici mogu učitati dokumente u različitim formatima (txt, html, pdf, docx, epub, itd.) ili uneti URL, nakon čega aplikacija prepoznaje teme i ključne reči unutar teksta. Tekstualni podaci se prvo obrađuju kroz proces normalizacije, uklanjanja stop reči i lematizacije. Zatim, LDA model grupiše reči u teme, a korisnici mogu prilagoditi broj tema i prolaza. Aplikacija takođe vizualizuje distribuciju reči po temama i omogućava dodeljivanje tema svakom dokumentu.
